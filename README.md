# Calculadora Goma Eva

Calculadora profesional para calcular costos y precios de trabajos en goma eva. Soporta 8 formas geométricas, márgenes personalizables y guardado de trabajos.

## Funcionalidades

- **8 formas geométricas** — Círculo, triángulo, cuadrado, pentágono, hexágono, octágono, rectángulo y entrada manual
- **Cálculo automático** — cm² según la forma seleccionada
- **Costos por ítem** — Materiales + mano de obra + gastos generales
- **Márgenes** — Precio mayorista y minorista configurables
- **Guardar trabajos** — Persistencia en localStorage (cargar/guardar presupuestos)

## Stack

- Vite + JavaScript (sin framework)
- Sin dependencias de UI — CSS propio

## Instalación

```bash
npm install
npm run dev
```
